# Один аккаунт = одна строка. Поля через пробел (или через двоеколонье).
# Форматы:
#   логин пароль
#   логин пароль 2fa-ключ(base32, если включена 2FA)
#   логин пароль 2fa-ключ прокси(host:port:user:pass или http://user:pass@host:port)
# Строки с # и пустые игнорируются.
#
# ВАЖНО: логиньтесь ЖЕЛАТЕЛЬНО через прокси в стране аккаунта — тогда сессия чистая
# и после импорта в приложение действия пойдут через тот же прокси без прыжка IP.
#
# Примеры (замените на свои, потом сохраните файл как accounts.txt):
my_account_1 SuperPass123
my_account_2 SuperPass456 OJDU3SXQHPJG7ABC
my_account_3 SuperPass789 OJDU3SXQHPJG7ABC 161.77.67.134:50100:login:pass
